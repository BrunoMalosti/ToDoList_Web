    Este aplicativo ToDo App foi desenvolvido com o objetivo de gerenciar tarefas pessoais de forma prática e intuitiva, permitindo criar, editar, concluir e excluir atividades, além de acompanhar prazos de maneira organizada.

    Desenvolvimento realizado por: Bruno Malosti e Marcio Ferre. O projeto foi idealizado e implementado utilizando HTML, CSS e JavaScript, seguindo boas práticas de programação para garantir uma interface amigável e responsiva.
